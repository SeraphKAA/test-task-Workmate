# test-task-Workmate (Анализатор эффективности разработчиков)
Тестовое задание от компании Workmate с работой csv файлов

Инструмент для анализа эффективности работы разработчиков на основе данных из csv файлов

## Установка (возможно потом уберу)

### Первый этап 

Клонирование репозитория и .env файла

```
git clone git@github.com:SeraphKAA/test-task-Workmate.git
cd app_for_FKW
```


### Второй этап

Создание окружения и загрузка библиотек из requirements.txt

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Использование

можно использовать как в примере тз, также можно использовать заготовленные команды написанные в Makefile

### Команды Makefile

- Запуск main файла, где в параметре передаем использование csv файла employees1.csv
```
make start
```

- тестирование и просмотр результатов (возможно тоже уберу, внеся рез-ы тестирования в readme)
```
make tests
```
